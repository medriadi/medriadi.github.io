Features

Create and manage tasks

Easily create, edit, and manage your tasks.

Set due dates and priorities

Organize your tasks by setting due dates and priorities.

Categorize tasks with labels

Use labels to categorize and find tasks quickly.

Mark tasks as complete

Check off tasks as you complete them.

Search and filter tasks

Find tasks quickly with search and filter options.

Get reminders and notifications

Never miss a task with reminders and notifications.

About This Project

Welcome to Smart-TODO! This project was inspired by the need for a simple and efficient task management application. It started as a personal project to improve productivity and has grown into a tool that can help others stay organized.

This project is part of the Portfolio Project for ALX.
